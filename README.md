### 制作アプリのタイトル

Atora（アトラ） | Path of Sages　さくらサーバー版

### 制作アプリの説明（40文字程度）

自然と調和し自己をチューニングする、貴方だけの旅のキュレーター・アーカイブ

- Concept / コンセプト
  Atoraは、単なる観光ガイドではなく、「貴方だけの旅のキュレーター」として機能します。
  現代人が抱える日常の悩み（自己・他者・生業）を、日本の歴史の中で自らを研鑽し、自然と深く対話してきた賢人・偉人たちの視点とマッチングさせます。彼らがその土地で何を感じ、どのように自然と調和したのか。そのプロセスを辿ることで、ユーザー自身の波長を整え（チューニング）、新たな気づきを得るための精神的な旅路をキュレーションします。

### 工夫した点・こだわった点

- 技術スタックの最適化:
  config.php 内で $\_SERVER['SERVER_NAME'] を利用した環境自動判別ロジックを実装。ローカル（MAMP等）と本番（さくらサーバー）の接続情報を書き換えることなくシームレスに開発・デプロイが可能な設計を追求。
- セキュアな認証（password_hash/password_verify）を用いたログイン機能の実装。
- デザインの静寂感:
  賢人の思考に触れる体験を壊さないよう、MontserratとNoto Serif JPを組み合わせたタイポグラフィを採用。
  Leaflet.jsを用いたマップデザインをモノトーン基調で統一し、静謐なUI/UXを実現。
- パーソナライズロジック:
  ユーザーが入力した悩みの系統をPHPで解析。最適な賢人ルートが地図上のリスト上部に優先表示されるレコメンドロジックを構築。
- 管理・運用の分離:
  一般ユーザーの「澱（悩み）の吐露」と、管理者の「賢人のアーカイブ登録」を admin_input.php として分離。

### 次回トライしたこと（または機能）

- **データの厚み**: 賢人の一言（Quote）や、現地で行う具体的な「3ステップの儀式（Ritual）」のカラムを追加し、コンテンツの解像度を高めたいです。
- **管理者画面の拡充**: 緯度経度や物語をより直感的に登録・編集できる専用のダッシュボード機能の構築。
- **感情分析**: ユーザーの悩みのテキストから感情を分析し、より精度の高いレコメンドエンジンを実装したい。

### 備考（感想、シェアしたいこと等なんでも）

PHPとデータベースを連携させることで、ユーザーの「今」の感情に呼応する動的な体験作りの可能性を実感しました。デプロイにおけるDB接続のトラブルシューティングを通じ、サーバー環境特有の挙動やセキュリティ、エラーハンドリングの重要性を深く学ぶことができました。
